# Razorpay UI Clone 💳

A responsive Razorpay landing page clone built using **HTML** and **Tailwind CSS**.  
This project focuses on clean UI, modern layout, and responsive design.

## 🚀 Features
- Modern fintech landing page UI
- Responsive navigation bar
- Tailwind CSS utility-first styling
- Clean and reusable components
- Mobile-friendly layout

## 🛠️ Tech Stack
- HTML5
- Tailwind CSS
